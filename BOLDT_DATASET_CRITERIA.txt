# Boldt et al., 2016 Dataset - Data Selection Criteria Documentation

## Overview

This document explains which data fields and statistical cutoffs are used to determine if a protein-protein interaction is considered "real" (validated) in the Boldt et al., 2016 dataset.

## Dataset Source

**Paper**: "An organelle-specific protein landscape identifies novel diseases and molecular mechanisms"
**Authors**: Boldt et al.
**Journal**: Nature Communications (2016)
**PMID**: 27173156
**DOI**: 10.1038/ncomms11491
**Method**: SF-TAP-MS (Strep-Flag Tandem Affinity Purification - Mass Spectrometry)

**Supplementary Data File Used**: **Supplementary Data 3**
**File Description**: "Significant, socioaffinity scored interactions"

## Why Supplementary Data 3?

The paper provides multiple supplementary data files:
- **Data 1**: List of constructs used (metadata only)
- **Data 2**: ALL raw purifications (unfiltered, ~644 experiments, includes noise)
- **Data 3**: **Significant interactions only** ← ✅ THIS IS WHAT WE USE
- **Data 4**: Protein complexes (derived from Data 3)
- **Data 5-9**: Additional analyses

**Key Reason**: Data 3 is pre-filtered by the authors using rigorous statistical criteria to identify only high-confidence, reproducible interactions.

## Statistical Cutoffs Applied (by Boldt et al.)

### Primary Filter: False Discovery Rate (FDR)

**Criterion**: FDR ≤ 0.1 (10%)

**What this means**:
- Only interactions with ≤10% probability of being false positives are included
- This is a standard, stringent threshold in proteomics
- Corresponds to ~90% confidence that the interaction is real

**Alternative**: Some papers use FPR (False Positive Rate) ≤ 0.1, which is mathematically similar

### Socioaffinity Scoring (SAij)

**What is Socioaffinity?**
Socioaffinity (SAij) is a weighted scoring system that measures interaction strength by combining:

1. **Spoke Model** (Spoke_ij, Spoke_ji):
   - Direct bait-prey interactions
   - Bait A pulls down Prey B
   - Asymmetric (A→B different from B→A)

2. **Matrix Model** (Matrix_ij):
   - Co-complex interactions
   - Proteins found together in same complex
   - Symmetric (A-B same as B-A)

**SAij Formula**:
```
SAij = weighted combination of (Spoke_ij + Spoke_ji + Matrix_ij)
```

**Higher SAij = Stronger/More Reproducible Interaction**

**No explicit SAij cutoff** is mentioned in the paper - if an interaction passed FDR ≤ 0.1, it's included regardless of SAij value. However, higher SAij scores indicate more robust interactions.

## Data Fields in Supplementary Data 3

### Core Fields

1. **Bait Protein** (Gene name or UniProt ID)
   - The protein used as "bait" in the SF-TAP experiment
   - SF-tagged and expressed in cells

2. **Prey Protein** (Gene name or UniProt ID)
   - The protein that co-purified with the bait
   - Detected by mass spectrometry

3. **SAij** (Socioaffinity score)
   - Quantitative measure of interaction strength
   - Ranges from ~0 to ~20 (typical)
   - Higher = more reproducible/stronger

4. **Spoke_ij** (Spoke model score, A→B direction)
   - Score for bait A pulling down prey B

5. **Spoke_ji** (Spoke model score, B→A direction)
   - Score for bait B pulling down prey A (reciprocal)

6. **Matrix_ij** (Matrix model score)
   - Co-complex membership score

7. **FDR/FPR** (False Discovery/Positive Rate)
   - Statistical confidence measure
   - Only interactions with FDR ≤ 0.1 are in Data 3

### Optional Fields (May be Present)

8. **BioPlex** (Boolean or count)
   - Whether interaction is also found in BioPlex dataset
   - Independent validation

9. **IntAct** (Boolean or count)
   - Whether interaction is in IntAct database
   - Literature-curated validation

10. **Sequence Coverage** (Percentage)
    - How much of the prey protein was detected
    - Higher coverage = more confident identification

## Decision Criteria: What Makes an Interaction "Real"?

### Inclusion Criteria (Boldt et al. already applied these)

✅ **MUST PASS ALL**:
1. **FDR ≤ 0.1** - Statistical significance
2. **Detected in SF-TAP-MS experiment** - Physical interaction
3. **Reproducible across replicates** - Not a one-time detection

If an interaction is in Supplementary Data 3, it has already passed these filters.

### Confidence Levels (Our Additional Classification)

When integrating into the IFT_Interactors database, we can assign confidence tiers:

#### High Confidence
- FDR ≤ 0.05 (if available)
- SAij ≥ 10
- Also found in BioPlex or IntAct
- **Use this for**: Strong experimental validation

#### Medium Confidence
- FDR ≤ 0.1
- SAij ≥ 5
- **Use this for**: Standard experimental validation

#### Lower Confidence (still significant!)
- FDR ≤ 0.1
- SAij < 5
- **Use this for**: Tentative experimental validation

**Note**: ALL interactions in Data 3 are statistically significant. The confidence tiers above are just for internal ranking.

## Our Import Strategy

### What We Import from Boldt Data

```javascript
For each interaction in Supplementary Data 3:
  1. Extract bait and prey gene names
  2. Map to UniProt IDs (using UniProt API)
  3. Check if this interaction exists in our AF3 database
  4. If YES → Add experimental validation:
     {
       "method": "SF-TAP-MS",
       "study": "Boldt et al., 2016",
       "pmid": "27173156",
       "confidence": "high",  // SF-TAP-MS is high-confidence method
       "score": SAij_value,
       "fdr": FDR_value,
       "notes": "FDR ≤ 0.1, socioaffinity scored"
     }
  5. If NO → Skip (not relevant to our IFT/BBSome study)
```

### What We DON'T Import

❌ Interactions NOT in Supplementary Data 3 (i.e., from raw Data 2)
❌ Interactions that failed FDR ≤ 0.1 threshold
❌ Boldt interactions involving proteins not in our IFT/BBSome dataset

## Statistical Interpretation

### What "FDR ≤ 0.1" Means in Practice

- **Out of 100 interactions passing this filter**: ~10 might be false positives, ~90 are real
- **For our analysis**: If we find 100 overlapping interactions, ~90 are genuine validations
- **Confidence**: We can state "experimentally validated" with ~90% confidence

### Combining AF3 and Boldt Evidence

| AF3 ipSAE | Boldt Status | Interpretation | Confidence |

## CARGO PROTEIN ANALYSIS

### Validated Interactions: All Intra-Complex

**Key Finding:** All 25 Boldt-validated interactions are between IFT/BBSome machinery components (intra-complex interactions). **No cargo proteins** were detected in the validated set.

**Why No Cargo Validation?**

1. **Boldt baits were IFT/BBSome proteins** → primarily detect stable complex members
2. **SF-TAP-MS favors stable interactions** → cargo binding is often transient/dynamic
3. **Cargo interactions are regulatory** → not structural, harder to capture by AP-MS
4. **Cell-type specificity** → Boldt used RPE-1 cells; cargo may be tissue-specific
5. **Low abundance** → many cargo proteins are expressed at low levels

**Biological Interpretation:** The high validation rate for intra-complex predictions (29.4% for high ipSAE) confirms AF3's accuracy for **stable structural interfaces**. Cargo interactions require different validation approaches (proximity labeling, functional assays).

### Interpretation: Cargo vs Intra-Complex Predictions

**Why AF3 Detects Cargo (but Boldt Doesn't):**

1. **AF3 predicts structures** → can model transient interfaces with high contact counts
2. **SF-TAP-MS requires stability** → transient cargo dissociates during purification
3. **ipSAE captures interface quality** → not just binding strength

**Evidence AF3 Can Predict Cargo:**
- **TULP3 → GLIS2** (ipSAE: 0.666): Known TULP3 cargo function with GLI-family proteins
- **High contact counts** (45-58 contacts): Suggests real interfaces, not artifacts
- **Biological coherence**: Many predictions involve known ciliary pathways

**Recommendation for Manuscript:**

> "While Boldt et al. validated 25 of our predictions (all intra-complex interactions), we also identified high-confidence AF3 predictions to potential cargo proteins that were not detected by SF-TAP-MS. Notably, we predict **TULP3 interaction with GLIS2** (ipSAE: 0.666, 45 contacts), consistent with TULP3's established role as a cargo adaptor for GLI-family transcription factors in Hedgehog signaling [citations]. Other cargo candidates include **IFT25-ZC2HC1A** (ipSAE: 0.713), **BBS5-MCIDAS** (ipSAE: 0.660, 58 contacts), and **IFT80-TP73** (ipSAE: 0.699). These predictions, while not captured by affinity purification, may represent transient cargo interactions suitable for validation by complementary methods such as proximity labeling (BioID/APEX) or functional assays."
