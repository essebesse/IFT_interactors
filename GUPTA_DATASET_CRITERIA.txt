## BioID Method Explained

### What is BioID?

**BioID** = Proximity-dependent Biotinylation Identification

**How it works**:
1. **Bait protein** is fused to **BirA* biotin ligase** (promiscuous mutant)
2. In presence of biotin, BirA* biotinylates **nearby proteins within ~10nm radius**
3. Biotinylated proteins are purified using **streptavidin beads**
4. Identified by **mass spectrometry**

**Key Features**:
- **Proximity labeling**: ~10nm radius (not necessarily direct binding)
- **In vivo**: Works in living cells (native context)
- **Transient interactions**: Captures weak/transient partners
- **Membrane proteins**: Works well for membrane-associated complexes

### BioID Confidence Level

**Confidence**: **Medium** (for our validation framework)

**Why medium, not high?**
- ✅ Proximity (~10nm) ≠ direct physical interaction
- ✅ Can label neighboring proteins without direct contact
- ✅ Higher false positive rate than TAP-MS/Co-IP
- ✅ BUT: Very useful for identifying functional neighbors and complexes

**Interpretation**:
- BioID hit = "Bait and prey are physically close in the cell"
- Does NOT prove direct binding
- Excellent for validating structural neighbors in IFT/BBSome trains

## Statistical Cutoffs Applied (by Gupta et al.)

### Primary Filter: SAINT Analysis

**SAINT** = Significance Analysis of INTeractome

**Criteria**:
1. **SAINT probability ≥ 0.8** (80% confidence)
2. **Fold-change vs control ≥ 2.0** (enriched over background)
3. **Detected in ≥2 biological replicates**

**What this means**:
- Only interactions with ≥80% probability of being real are included
- Must be at least 2× enriched over negative controls
- Must be reproducible across experiments

### Spectral Counts

**Average Spectral Count (AvgSpec)**:
- Number of MS/MS spectra identifying the prey protein
- Higher AvgSpec = more abundant interaction
- **High confidence**: AvgSpec ≥ 10
- **Medium confidence**: AvgSpec 5-10
- **Lower confidence**: AvgSpec 2-5

### Control Comparisons

**Negative controls** used:
1. **Empty vector** (no BirA* fusion)
2. **Non-specific BirA*** (control localization)
3. **Non-ciliated cells** (HEK293T without cilium induction)

Only proteins significantly enriched over ALL controls are included.

## Data Fields in Table S1

### Core Fields

1. **Bait Protein** (Gene name or UniProt ID)
   - Protein fused to BirA* biotin ligase
   - Localized to centrosome, transition zone, or cilium

2. **Prey Protein** (Gene name or UniProt ID)
   - Biotinylated protein identified by MS
   - Proximity interactor within ~10nm

3. **SAINT Probability** (0-1 scale)
   - Statistical confidence score
   - ≥0.8 for high confidence (standard cutoff)
   - ≥0.9 for very high confidence

4. **Average Spectral Count (AvgSpec)**
   - Mean number of MS/MS spectra across replicates
   - Proxy for interaction strength/abundance

5. **Fold-Change (FC)**
   - Enrichment over negative controls
   - FC ≥ 2.0 typically required
   - Higher FC = more specific interaction

6. **Localization** (Centrosome/Transition Zone/Cilium)
   - Where the bait protein localizes
   - Important for interpreting spatial relationships

### Optional Fields (May be Present)

7. **Biological Replicates** (Count)
   - Number of experiments detecting this interaction
   - More replicates = higher confidence

8. **Control Spectral Count** (CtrlSpec)
   - Background signal in controls
   - Lower is better (less non-specific)

9. **Known Interaction** (Boolean)
   - Whether interaction was previously reported
   - Validates BioID method accuracy

## Decision Criteria: What Makes an Interaction "Real"?

### Inclusion Criteria (Gupta et al. applied these)

✅ **MUST PASS ALL**:
1. **SAINT probability ≥ 0.8** - Statistical significance
2. **Fold-change ≥ 2.0** - Specific enrichment over controls
3. **Detected in ≥2 replicates** - Reproducibility
4. **AvgSpec ≥ 2** - Minimum detection threshold

If an interaction is in Table S1, it has already passed these filters.

### Confidence Levels (Our Additional Classification)

When integrating into the IFT_Interactors database, we can assign confidence tiers:

#### High Confidence BioID
- SAINT probability ≥ 0.9
- AvgSpec ≥ 10
- Fold-change ≥ 5
- Detected in all replicates
- **Use this for**: Strong proximity validation

#### Medium Confidence BioID
- SAINT probability ≥ 0.8
- AvgSpec ≥ 5
- Fold-change ≥ 2
- **Use this for**: Standard BioID validation

#### Lower Confidence BioID
- SAINT probability ≥ 0.7 (if included)
- AvgSpec 2-5
- Fold-change ≥ 2
- **Use this for**: Tentative proximity association

**Note**: ALL interactions in Table S1 are statistically significant. The tiers above are just for internal ranking.

## Our Import Strategy

### What We Import from Gupta Data

```javascript
For each interaction in Table S1:
  1. Extract bait and prey gene names
  2. Map to UniProt IDs (using UniProt API + cache)
  3. Check if this interaction exists in our AF3 database
  4. If YES → Add experimental validation:
     {
       "method": "BioID",
       "study": "Gupta et al., 2015",
       "pmid": "26638075",
       "confidence": "medium",  // BioID is proximity, not direct binding
       "bait_protein": "CEP290",
       "score": SAINT_probability,
       "spectral_count": AvgSpec,
       "fold_change": FC,
       "notes": "Proximity labeling (~10nm), centrosome-cilium interface"
     }
  5. If NO → Skip (not relevant to our IFT/BBSome study)
```

### What We DON'T Import

❌ Interactions with SAINT probability < 0.8 (not in Table S1)
❌ Interactions failing fold-change cutoff
❌ Non-reproducible interactions (single replicate only)
❌ Gupta interactions involving proteins not in our IFT/BBSome dataset

## Comparison: BioID vs TAP-MS (Boldt)

| Feature | Boldt (TAP-MS) | Gupta (BioID) |
|---------|----------------|---------------|
| **Method** | Affinity purification | Proximity labeling |
| **Interaction type** | Direct physical binding | Proximity (~10nm) |
| **Confidence** | High | Medium |
| **False positives** | Low | Moderate |
| **Transient interactions** | Often missed | Captured well |
| **Membrane proteins** | Can be lost | Work well |
| **Stoichiometry** | Strong binders favored | All neighbors labeled |
| **Our use** | Primary validation | Complementary validation |

**Synergy**:
- Boldt + Gupta overlap = **Very high confidence** (direct + proximity)
- Boldt only = High confidence (direct interaction)
- Gupta only = Medium confidence (proximity/functional association)
- Neither = Novel AF3 prediction

## Statistical Interpretation

### What "SAINT ≥ 0.8" Means in Practice

- **Out of 100 interactions passing this filter**: ~20 might be false positives, ~80 are real proximities
- **For our analysis**: Lower stringency than Boldt (FDR ≤ 0.1 = ~10% FP)
- **Confidence**: We can state "proximity validated" with ~80% confidence
